# SSHCopy
Ferramenta gráfica para utilização do SCP via ssh para linux.
# Instalação: 
<p>git clone https://github.com/leonidasroberto/SSHCopy</p>
<p>cd SSHCopy</p>
<p>sudo chmod 755 SSHCopy</p>
<p>sudo apt install sshpass</p>
<p>Usar !!!<p/>
